Restaurant Website

What is this site

This was the first website Jonathan produced. It was about a restaurant called the best restaurant around. The overall goal was to present a restaurant themed website. It was designed to cater towards the many customers that would visit this site such that they would visit the establishment.

What was gained from making this site

The content reflects a actual restaurant website. There are 4 main pages. The first is a home page that has all the basic information from specialty dishes to descriptive information about the establishment's atmosphere. The specials page shows one daily special served at the establishment. The third page shows a performer who will be performing at the establishment. The last page shows a table of all future performers and the relevant dates.

This was the first website created, thus it has many beginner techniques involved in the color scheme and overall design. As he applied more html and css skills, Jonathan was able to improve the visual assets and also fix many long standing bugs in the way images wrapped the text.

Final Project Website

What is this site

This was the second website Jonathan produced. It was a Age of Empires 3 modding website. It was meant to act as a database for future modders of the pc game Age of Empires 3. This site was meant to be more insightful to newcomers than other sites with different features that would help tailor help to a individual rather than a crowd.

What this site contains

There are really two categories of pages that compose this site. One type are information pages such as the homepage, the info page, the FAQ, and the flag making tutorial. The other type are form pages such as the mod specifications form and the background form which were meant to be surveys to the user to be mailed to the website's server allowing developers to tailor tutorials and summaries for individuals rather than larger groups of modders. Of course the site would have general modding information pages like the flag page, but the forms were there to make the site cater to all people interested in the topic as well as others.

What was gained from making this site

This site was designed using more images than typical. The site design came together nicely compared to the previous site. The site itself in this situation was a product meant to educate people. Thus, the site was not only meant to be informational, but also interactive with the user with tools to help them in their endeavors. Writing the text content and finding good images was quite simple as Jonathan was familiar with the subject. The site is overall very informational.
